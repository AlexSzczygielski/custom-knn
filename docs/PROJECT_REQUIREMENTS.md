# Project Requirements

1. **Programming Language & Libraries**
   - Python 3.x
   - Libraries:
     - `numpy`
     - `matplotlib`
     - `scikit-learn` (for datasets, comparison, and metrics)

2. **Code Structure**
   - Separate modules for:
     - `knn.py` – Custom K-NN implementation
     - `test.py` – Testing and visualization routines

3. **Usability**
   - Easy-to-use interface through the `TestKNN` class.
   - `TestKNN` prepared to work with scikit datasets.
   - Ability to change parameters:
     - Number of neighbors (`k`)
     - Dataset split parameters:
        - Train/test split ratio
        - Random seed
        - Sample index for visualization

4. **Documentation**
   - All classes and methods contain respective docstrings.
   - Project includes usage examples in `test.py`.
